A book about identity as a service (IDaaS).

一本关于身份即服务的书，翻译和汇总 IAM、IDaaS、OAuth2 等相关专业知识，介绍相关软件。

## Roadmap

- IDaaS
- Keycloak
- OAuth2
- OIDC
- CAS
- Dex
- LDAP/AD/kerberos
- SAML
