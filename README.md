A book about identity as a service (IDaaS).

## Development

基于Hugo搭建，主题使用 hugo-geekdoc.

```bash
 # 增加新章节
 hugo new folder/name.md
 # 本地启动 Server
 hugo server -D
 # 发布
 hugo
```
