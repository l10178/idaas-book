  • Composite Rolesпјҡз»„еҗҲи§’иүІпјҲеҢ…еҗ«е…¶д»–и§’иүІпјү

з»“еҗҲ Group е®һзҺ°з”ЁжҲ· вҶ’ з»„ вҶ’ и§’иүІзҡ„й—ҙжҺҘжҳ е°„гҖӮ

  • и§’иүІзҲҶзӮёпјҡеҪ“дёҡеҠЎзІҫз»ҶеҢ–ж—¶пјҢи§’иүІж•°йҮҸжҝҖеўһ
  • йқҷжҖҒжҖ§пјҡи§’иүІдёҚиғҪж №жҚ®дёҠдёӢж–ҮпјҲж—¶й—ҙгҖҒдҪҚзҪ®гҖҒи®ҫеӨҮпјүеҠЁжҖҒи°ғж•ҙ
  • и·Ёеҹҹеӣ°йҡҫпјҡдёҚеҗҢзі»з»ҹзҡ„и§’иүІдҪ“зі»ж— жі•дә’йҖҡ

20.3 ABACпјҲеҹәдәҺеұһжҖ§зҡ„и®ҝй—®жҺ§еҲ¶пјү

ж ёеҝғжЁЎеһӢ

дёҚдҫқиө–и§’иүІпјҢиҖҢжҳҜж №жҚ®еӨҡз»ҙеәҰеұһжҖ§еҠЁжҖҒеҶіе®ҡпјҡ

XACML ж ҮеҮҶ

XACMLпјҲeXtensible Access Control Markup LanguageпјүжҳҜ ABAC зҡ„ж ҮеҮҶеҢ–е®һзҺ°жЎҶжһ¶пјҢеҹәдәҺ OASIS ж ҮеҮҶпјҡ

XACML иҷҪз„¶жҰӮеҝөејәеӨ§пјҢдҪҶеӨҚжқӮеәҰй«ҳгҖҒжҖ§иғҪејҖй”ҖеӨ§гҖӮеңЁж–°йЎ№зӣ®дёӯпјҢжӣҙеӨҡйҖүжӢ© OPAпјҲOpen Policy AgentпјүжҲ–еҹәдәҺзӯ–з•Ҙеј•ж“Һзҡ„иҪ»йҮҸж–№жЎҲгҖӮ

OPAпјҲOpen Policy Agentпјү

OPA жҳҜ CNCF жҜ•дёҡйЎ№зӣ®пјҢдҪҝз”Ё Rego иҜӯиЁҖзј–еҶҷзӯ–з•Ҙпјҡ

ABAC зҡ„дјҳеҠҝдёҺжҢ‘жҲҳ

дјҳеҠҝпјҡ

  • жһҒеәҰзҡ„зҒөжҙ»жҖ§пјҢж”ҜжҢҒеӨҚжқӮзӯ–з•Ҙ
  • дёҚдҫқиө–йў„е®ҡд№үи§’иүІ
  • ж”ҜжҢҒеҠЁжҖҒдёҠдёӢж–Ү

жҢ‘жҲҳпјҡ

  • зӯ–з•Ҙйҡҫд»ҘеҸҜи§ҶеҢ–е’Ңе®Ўи®ЎпјҲ“иҝҷдёӘз”ЁжҲ·жңҖз»ҲиғҪеҒҡд»Җд№Ҳпјҹ"пјү
  • зӯ–з•Ҙзј–еҶҷе’Ңз»ҙжҠӨжҲҗжң¬й«ҳ
  • жҖ§иғҪејҖй”ҖпјҲжҜҸдёӘиҜ·жұӮйңҖиҰҒиҜ„дј°еӨҡдёӘеұһжҖ§пјү
  • еұһжҖ§жқҘжәҗзҡ„еҸҜйқ жҖ§й—®йўҳ

20.4 ReBACпјҲеҹәдәҺе…ізі»зҡ„и®ҝй—®жҺ§еҲ¶пјү
