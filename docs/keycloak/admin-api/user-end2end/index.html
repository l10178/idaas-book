常见问题#

Q: 创建用户后收不到密码重置邮件？ 检查 Realm Settings > Email 中 SMTP 配置是否正确，测试连接后再试。

Q: 批量导入用户？ 使用 Keycloak 的 Partial Import 功能，或直接通过 Admin API 循环调用创建接口。

Q: 用户属性（attributes）怎么设？ 在创建请求的 attributes 字段中传 JSON 对象，例如 {"department":"engineering","region":"cn"}。
